# SamatNext Speed8

This repository tracks the active SamatNext-Speed-8L-56M throughput model and
the next real-data stability check.

Current scope:

- Synthetic training throughput verification for the active 56M all-softmax/GQA model.
- A prepared 1000-step real-data stability, loss, and perplexity check.

This repo does not claim that the model is a good coder yet. The current work
verifies training throughput and prepares real-data stability testing.
